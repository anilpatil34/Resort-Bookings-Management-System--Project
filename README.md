# 🏨 Resort Management System – Django CRUD Project

A complete Resort Management System built with **Python Django**, focusing on CRUD operations and basic resort workflow features such as check-in, check-out, and billing reports.

---

## 🚀 Features

### 🔹 Customer Management
- Add new customers  
- Update existing customer details  
- Delete customers  
- View all customer records  

### 🔹 Room Management
- Add new rooms  
- Update room information  
- Delete rooms  
- View rooms list  

### 🔹 Resort Operations
- Customer check-in  
- Customer check-out  
- Search customers or rooms  
- Billing report generation  

## 🛠️ Technologies Used
- Python  
- Django  
- HTML  
- CSS  
- Bootstrap  
- SQLite/MySQL (based on your setup)
